# mdcss theme boilerplate

1. To begin, [fork this project].

2. Next, clone your fork locally.
   ```sh
   git clone git@github.com:YOUR_NAME/mdcss-theme mdcss-theme-THEME_NAME
   ```

3. Finally, run npm install from your clone:
   ```sh
   npm install
   ```
